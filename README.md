# 🥊 METZ FIGHTER — Web Demo

Jogo de luta 2D no navegador, com os sprites dos personagens e o background da Metz.

## 🎮 Controles

| Tecla | Ação |
|-------|------|
| `J` | Soco |
| `K` | Chute |
| `L` (segurar) | Bloquear |
| `SPACE` | Pular |
| `← / A` | Mover Esquerda |
| `→ / D` | Mover Direita |
| `S` + `D` + `J` | **Golpe Especial** |
